SoC Classic Crash Fix

Disables ASLR for com.vnlentertainment.socclassic only via personality(ADDR_NO_RANDOMIZE) in Zygisk preAppSpecialize — so the game's heap is less likely to land past Unity 2017.4's ~16GB address ceiling. See root cause.

Preflight

Requirements

  • Rooted Android device
  • Zygisk-compatible root (KernelSU, or Magisk with Zygisk enabled)
  • arm64-v8a (64-bit) — matches the game's APK
  • School of Chaos Classic installed: com.vnlentertainment.socclassic
Procedure

Install steps

  1. Download socfix_module.zip from the package above.
  2. Open your root manager → Modules → Install from storage → pick the zip.
  3. Reboot the device.
  4. Launch the game. Optional: confirm the hook with logcat filter SocFix — you should see ASLR disabled for the package.

Limitations

Known caveats

  • May reduce crash frequency rather than eliminate it — one confirmed failure with mitigation active.
  • Primarily tested on KernelSU + Pixel 9 Pro XL; other OEMs are untested for the module itself.
  • The crash is confirmed on Transsion and Xiaomi families without any workaround installed — see confirmed devices.
  • Does not modify the game APK; only an OS-level process flag for that package name.
